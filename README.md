# Afk-Arena-Airtest 剑与远征的自动化脚本

### 请堆满你的暴击和闪避！

> Script 1 Battle-Forward：自动挑战首领。失败重试，胜利自动下一关。

> Script 2 King-Tower：王者之塔。失败重试，胜利自动挑战下一层。
